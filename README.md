# configuration-development-2

Dependency Graph Visualizer

Этапы разработки

Этап 1: Конфигурация
CLI-приложение с парсингом аргументов

Валидация параметров и обработка ошибок

Вывод конфигурации в формате ключ-значение

Этап 2: Сбор данных
Работа с NuGet API v3

Парсинг .nuspec файлов для получения зависимостей

Вывод прямых зависимостей пакета

Этап 3: Построение графа
Алгоритм BFS с рекурсией для транзитивных зависимостей

Ограничение глубины и фильтрация пакетов

Обнаружение циклических зависимостей

Поддержка тестовых репозиториев (A->B C)

Этап 4: Обратные зависимости
Режим поиска пакетов, которые зависят от целевого

BFS алгоритм для обратного анализа

Построение полного графа зависимостей

Этап 5: Визуализация
Генерация Graphviz DOT описания графа

Вывод кода на экран и сохранение в .dot файлы

Стилизация графов с подсветкой корневого пакета

Поддержка визуализации прямых и обратных зависимостей
